Spaced Repetition (SM-2) สำหรับนิสิตแพทย์ / สัตวแพทย์ — จำน้อยลง แต่นาน

ทำไมเด็กแพทย์ทั่วโลกใช้ Anki · ทำไม USMLE prep ทุก resource มี SR built-in · ทำไม "อ่านวันละนิด" ดีกว่า "ติวคืนเดียว 8 ชั่วโมง" — บทความนี้คือคำอธิบายแบบไม่ต้องโม้

ปัญหา: Forgetting Curve

Hermann Ebbinghaus ค้นพบในปี 1885 (140 ปีที่แล้ว) ว่ามนุษย์เรา ลืม 50% ของข้อมูลใหม่ภายใน 1 ชั่วโมง · 70% ภายใน 24 ชั่วโมง · 90% ภายใน 1 สัปดาห์ — ถ้าไม่ได้ทบทวนเลย

ลอง imagine: เรียน Cushing's syndrome วันอังคาร · สอบวันเสาร์ · ถ้าไม่ทบทวนเลย วันสอบจะเหลือ ~10-20% ของที่อาจารย์สอน · ไม่พอจะตอบข้อสอบเลย

หลักการ Spaced Repetition: ทบทวน "ตอนกำลังจะลืม" — ไม่ใช่ตอนยังจำได้ดี หรือลืมไปแล้ว · timing สำคัญ

SM-2 Algorithm — หลักคิดของ Anki

Piotr Wozniak พัฒนา SuperMemo-2 (SM-2) algorithm ปี 1987 · เป็น base ของ Anki, RemNote, Quizlet, และอีกหลายแอป · concept ง่ายมาก: 

1. แสดง flashcard / question
2. คุณตอบ → rate ความยาก: Again / Hard / Good / Easy
3. SM-2 คำนวณ "อีกกี่วันค่อยเจอข้อนี้อีก"
   - Again  → เจออีก 1 นาที (ลืมแล้ว)
   - Hard   → เจออีก ~6 วัน (ขอเตือน)
   - Good   → เจออีก ~10 วัน (ปกติ)
   - Easy   → เจออีก ~16 วัน (ห่างได้)
4. ถ้าตอบถูก น่ารัก ครั้งหน้าห่างขึ้นเรื่อยๆ
   - Round 1: 1 วัน
   - Round 2: 3 วัน
   - Round 3: 1 สัปดาห์
   - Round 4: 2 สัปดาห์
   - Round 5: 1 เดือน
   - ... จนข้อนี้ "stable" ใน long-term memory

ผลลัพธ์: ข้อที่ ง่าย/เคยจำได้แล้ว = เจอน้อย · ข้อที่ ยาก/ลืมง่าย = เจอบ่อย · เวลาที่คุณใช้ทบทวน กระจายไปยังจุดอ่อน โดยอัตโนมัติ

ทำไม Cramming คืนสอบไม่ work

Massed practice (cramming คืนเดียว 8 ชั่วโมง) สร้าง short-term memory ที่ใช้ได้แค่วันสอบ · พอจบสอบ 1 สัปดาห์ ลืมหมด · เจอข้อสอบใหม่ที่ใช้เนื้อหาเดิม = blank

Distributed practice (15-30 นาที / วัน × 14 วัน) ใช้เวลารวมเท่ากัน แต่ encode เข้า long-term memory ได้ลึกกว่ามาก · กลับไปสอบ comprehensive หรือ Comprehensive ในปีอื่น ก็ยังตอบได้

วิธีใช้กับข้อสอบสัตวแพทย์

1. ใช้กับ MCQ ที่ตอบผิด

ทุกครั้งที่ทำ mock exam → ข้อที่ผิด → mark เข้า SR system · พรุ่งนี้ระบบจะดันข้อนั้นกลับมาให้ลองอีก · ตอบถูก = เลื่อนต่อ · ตอบผิดอีก = ลดเวลาลง (เจอเร็วขึ้น)

2. ใช้กับคอนเซปต์ใน summary คลิป

สรุปคลิป Cushing → มี Exam Hot Spots 10 ข้อ · เปลี่ยนเป็น flashcard ลงใน SR · ภายใน 2-3 สัปดาห์ จะ "stick" โดยไม่ต้องตั้งใจท่อง

3. เริ่มเร็ว — ไม่ใช่ก่อนสอบ 3 วัน

Critical: SR ใช้ time-decay algorithm · ถ้าเริ่มก่อนสอบ 3 วัน = ไม่ได้รอบ "round 3 / round 4" ที่จำเป็นสำหรับ long-term encoding · ผลคือเสมอ cramming

เริ่มเลย — วันแรกที่อาจารย์สอน · เพิ่ม flashcard 5-10 ข้อ ทุกครั้งหลังเรียน · ใช้ 10-15 นาที / วันทบทวน · ก่อนสอบจะรู้สึก "ตื่นเช้ามาก็ครบแล้ว"

VetMock มี SR built-in

ใน VetMock ทุกข้อ MCQ ที่ทำผิดจะถูกเพิ่มเข้า SR queue อัตโนมัติ · ใน home dashboard จะเห็น "X cards due" · กดเปิด → ทำ session ทบทวน 5-10 นาที

Algorithm ใช้ SM-2 ดัดแปลง (ดู src/hooks/sm2.js ใน open-source repo) · Bookmark + custom questions ก็เข้า SR ได้เช่นกัน

เริ่มยังไง

  1. วันนี้ — ลงทะเบียน VetMock (ฟรี) · เลือกวิชาที่กำลังเรียนอยู่
  2. ทำ quiz 10 ข้อ — ข้อที่ผิด จะเข้า SR queue อัตโนมัติ
  3. ทุกวัน 5-10 นาที — เปิดดู "cards due" · ทำ session ทบทวน
  4. 1 สัปดาห์หลัง — สังเกต: ข้อที่เคย stuck เริ่ม "เห็นแว่บเดียวก็ตอบได้"
  5. ก่อนสอบ — Mock exam full simulation · จะเห็น improvement ชัดเจน
