เริ่มจากปัญหาเล็กๆ ที่นิสิตปี 4 ทุกคนเจอ — "ข้อสอบเก่าหายไปไหนหมด?" — จนกลายเป็นระบบรวมศูนย์ที่ตอนนี้มี MCQ + ข้อสอบเก่า + คลิปย้อนหลัง + summary ครอบคลุม COM III / IV / V + Eng Vet Prof II

ทำไมต้องสร้างเอง?

ก่อนหน้านี้ข้อสอบเก่าของ นิสิตสัตวแพทย์ จุฬา กระจายอยู่ตาม LINE group, Google Drive, ใต้กระดาษเลกเชอร์รุ่นพี่ บางคนได้บางวิชาแต่ไม่ได้บางวิชา · บางข้อตอบไม่ตรงกัน 2 รุ่นเพราะมีคนแก้ไม่ครบ · บางคลิปอาจารย์มีรุ่นพี่ดูสรุปเก็บไว้ แต่ไม่มีระบบหาง่าย

VetMock เริ่มจากความคิดง่ายๆ — "ถ้ามีที่เดียวเก็บ + ค้นหาง่าย + อยู่ทุกอุปกรณ์ ก็ดี" เลยลงมือสร้าง progressive web app (PWA) ที่:

  • เก็บ MCQ + ข้อสอบเก่า เป็น single source of truth — แก้ครั้งเดียว ทุกคนเห็นใหม่
  • ทำ mock exam จับเวลา + คะแนน + เห็น mistake เพื่อกลับมาทบทวน
  • Bookmark + spaced repetition — ข้อที่ผิดบ่อยจะเด้งขึ้นมาอีก
  • Summary คลิปอาจารย์ — อ่าน 10 นาทีแทนดู 120 นาที (สำหรับ revision)
  • ตารางสอบรวมศูนย์ — มีตารางสอบของ Vet 86 ปี 4 ครบ
  • ทำงาน offline ในมือถือ (PWA · install ลง home screen ได้)

ไม่ใช่ระบบของอาจารย์ — เป็นของนักเรียน

VetMock ไม่ได้ เกี่ยวข้องกับคณะหรือมหาวิทยาลัย · เป็นโปรเจค side project ของนิสิตในรุ่นที่อยากแบ่งปันเครื่องมือเตรียมสอบให้เพื่อนๆ

ทุกข้อสอบที่เพิ่ม — มาจากการลองทำเอง + ดูเฉลย + ทดสอบกับเพื่อน ไม่ได้ดูดมาจากอาจารย์โดยตรง · summary ของคลิปก็ทำขึ้นเพื่อใช้เตรียมสอบ ไม่ได้แทนที่การฟังคลิปจริง (ฟังคลิปยังต้องฟังนะ — มี nuance ที่ summary จับไม่ได้)

กำลังขยายให้ครอบคลุมทุกชั้นปี

ตอนนี้เน้นปี 4 ก่อน เพราะคนทำเป็นปี 4 · แผนระยะยาวคือ ปิดเทอมหน้า จะเริ่มทำของชั้นปีอื่นๆ:

  • ปี 1-2 — Pre-clinic (Anatomy, Physiology, Histology, Biochem)
  • ปี 3 — Paraclinic (Pathology, Pharmacology, Microbiology, Parasitology)
  • ปี 4 — ✅ COM III / IV / V + Eng Vet Prof II (พร้อมใช้)
  • ปี 5-6 — Clinical rotation + Internship

เป้าหมาย: ครบทุกชั้นปี (1-6) ภายใน 1-2 ปี โดยอาศัยรุ่นน้องช่วยคอนเฟิร์มความถูกต้องของแต่ละวิชา

คำถามที่ถามบ่อย

ฟรีจริงไหม?

ฟรี · ไม่มีโฆษณา · ไม่มีพรีเมียม · ไม่ขายข้อมูล (เพราะคนทำเป็นนิสิตสัตวแพทย์ ไม่ใช่ startup)

ทำไมไม่ใช่แอป iOS / Android?

เป็น PWA (Progressive Web App) — ติดตั้งบนมือถือได้ โดยกด "Add to Home Screen" ใน Safari / Chrome · เปิดแอปได้ offline ทำเป็น native app ก็ได้แต่ดูแลยากกว่า · PWA แก้ครั้งเดียวทุกคนเห็นใหม่

ใช้นอกจุฬาได้ไหม?

ใช้ได้ครับ — เปิดให้ทุกคน แต่ content ตอนนี้อิงตาม syllabus ของจุฬาฯ ถ้าเป็นนิสิตจากมหาวิทยาลัยอื่น ข้อสอบบางส่วนอาจไม่ตรง syllabus
